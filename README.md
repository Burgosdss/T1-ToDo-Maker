# Set Up
1. Download the Code
2. Extract the folder from the .zipfile and cdinto it
3. Install the Node modules: $ npm i
4. Open the code in VS Code: $ code .
5. Start the dev server: $ npm start  
6. Back server: $ nodemon server

# Technologies Used
* Mongo.
* Express.
* React.
* Nodejs.
* JWT for Authentication.
